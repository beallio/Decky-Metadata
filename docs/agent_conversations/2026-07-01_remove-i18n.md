# 2026-07-01 remove-i18n

## Objective

Remove the frontend i18n layer and inline the existing English strings without changing user-visible behavior.

- `src/index.tsx`
- `src/contextMenuPatch.tsx`
- `src/components.tsx`
- `src/steam.ts`

## Design Decisions

- Replaced 195 literal `t("key")` / `t('key')` call sites with JSON-encoded values from `STRINGS.en`.
- Converted the achievement cache and achievement source dynamic-key call sites to module-scope English label maps.
- Converted RetroAchievements resolution messages to return the same English message text directly.
- Deleted `src/i18n.ts` and removed all `./i18n` imports.
- Left runtime behavior unchanged aside from removing language switching; English text is the preserved source of truth.
